## 쿼리문 수행 순서
+ select -> from -> where -> group by -> having -> column -> order by -> limit

## 쿼리문 문장 종류
+ DML : 데이터 조작어 [SELECT, INSERT, UPDATE, DELETE]
+ DDL : 데이터 정의어 [CREATE, ALTER, DROP, TRUNCATE]
+ DCL : 데이터 제어어 [GRANT, REVOKE, COMMIT, ROLLBACK]
